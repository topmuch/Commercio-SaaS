# Task 4.3 — Create /api/ai/route.ts with real LLM + update ai-assistant-page.tsx

## Work Done

### Part 1: Created `/src/app/api/ai/route.ts`
- **POST handler** accepting `{ question: string, history?: Array<{role, content}> }`
- **Auth check**: `getServerSession(authOptions)` → 401 if no session
- **Company context fetched in parallel** (6 queries via `Promise.all`):
  - Total clients count (`db.client.count`)
  - Revenue this month (`db.order.aggregate` with `_sum.total`, filtered by `createdAt >= monthStart`)
  - Top 3 products by revenue (`db.orderItem.groupBy` by `productId`, ordered by `_sum.totalPrice desc`)
  - Top 3 commercials by revenue (`db.order.groupBy` by `commercialId`)
  - Active orders count (status in: new, validated, preparation, shipped)
  - Stock alerts count (products where `stock < minStock`)
- **Follow-up queries** to resolve product names and commercial names from groupBy IDs
- **System prompt in French** with all context data injected
- **LLM call** via `z-ai-web-dev-sdk`: `ZAI.create()` → `zai.chat.completions.create()` with `thinking: { type: 'disabled' }`
- **Error handling**: try/catch with 401/400/500 status codes and descriptive error messages
- **Helper function**: `formatCFA()` for French-style number formatting (space separators)
